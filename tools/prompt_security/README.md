\# Prompt Security Framework



This module provides AI prompt safety validation and injection detection tooling.



\## Features



\- Prompt injection detection

\- Jailbreak pattern scanning

\- System prompt extraction detection

\- Prompt override detection



\## Usage



Run:



```bash

python tools/prompt\_security/injection\_detector.py

```



Run tests:



```bash

pytest tools/prompt\_security/test\_injection\_detector.py

```
